# 03. Platzi e-commerce

Funcionalidad básica de un e-commerce:

* Lista de productos
* Agregar a carrito
* Lista de ordenes de compra
* Buscador por nombre de producto
* Filtro por categorías

Para correrlo descarga el proyecto y ejecuta

`npm i`

Finalmente ejecuta

`npm run dev`
